# 05 — Live 90-Minute Session

## Goal

Transform the session from:

```text
normal language lesson
```

into:

```text
DTZ combat training session
```

---

# Session Stack

```text
Google Meet / Zoom
+ Google Docs Live Worksheet
+ Cowork Context Pack
+ Tutor Notes
+ Optional recording/transcript
```

---

# Standard Speaking Session

## Timeline

```text
00:00–00:10 Warmup
00:10–00:25 Chunk Drill
00:25–00:50 DTZ Simulation
00:50–01:05 Error Mining
01:05–01:20 Retry Loop
01:20–01:30 Reflection + Save
```

---

# Warmup

Goal:
- activate German mode
- reduce speaking freeze
- build flow

Tutor should:
- ask simple daily questions
- avoid heavy correction
- keep student speaking

---

# Chunk Drill

Use:
- required chunks
- survival phrases
- opinion patterns
- clarification phrases

Examples:

```text
Meiner Meinung nach ...
Das kommt darauf an.
Könnten Sie das bitte wiederholen?
Ich bin nicht sicher, aber ...
```

Goal:
- automatic output
- speaking muscle memory

---

# DTZ Simulation

Examples:

- roleplay
- Bildbeschreibung
- Planen
- timed writing
- listening traps
- reading scan

Rules:

- tutor should not interrupt too much
- errors should be noted
- simulation should feel real

---

# Error Mining

Tutor + Cowork identify:

- recurring grammar errors
- hesitation moments
- weak chunks
- pronunciation problems
- listening failures
- reading mistakes

Only prioritize high ROI problems.

---

# Retry Loop

Student repeats:

- corrected roleplay
- corrected sentences
- corrected writing
- corrected listening task

Goal:

Immediate neural reinforcement.

---

# Reflection + Save

Required outputs:

```text
- worksheet complete
- error list
- retry drills
- strong chunks
- weak grammar
- homework
- next-session suggestions
```

If possible:

- recording
- transcript
- speaking notes
