# 00 — System Overview

## Big Picture

```text
INPUT DATA
PDF / Google Docs / Audio / Transcript / Rubric / Tutor Notes
        ↓
CLAUDE CODE + COWORK PROCESSING
Extract patterns, chunks, templates, traps, grammar targets
        ↓
KNOWLEDGE BASE
Structured DTZ database in markdown / Notion
        ↓
SESSION GENERATION
Cowork creates personalized 90-minute session
        ↓
LIVE SESSION
Tutor + student + interaction platform
        ↓
SESSION OUTPUT
Worksheet + transcript + errors + retry drills
        ↓
PERSONALIZATION LOOP
Student profile becomes smarter
```

## Main Roles

### Student

Creates output:
- speaking
- writing
- listening answers
- reading scan
- retry drills

### Tutor

Execution layer:
- simulate examiner
- pressure interaction
- note recurring errors
- avoid over-teaching grammar
- maintain speaking flow

### Cowork

Session orchestrator:
- create session packs
- choose chunks
- choose grammar targets
- generate retry drills
- track student profile

### Claude Code

Processing layer:
- ingest PDFs/docs/audio
- extract patterns
- generate markdown files
- update databases
- automate reports
- generate Anki CSV

## DTZ Training Philosophy

| Skill | Philosophy |
|---|---|
| Sprechen | Performance + interaction |
| Schreiben | Template assembly |
| Hören | Keyword extraction |
| Lesen | Scanning + matching |

DTZ should not be trained as general German.
