# Brainstorm

Thư mục này lưu các ý tưởng đã thống nhất khi xây dựng hệ thống học tiếng Đức dài hạn.

Mục tiêu của thư mục này:

- lưu lại các ý tưởng lớn trước khi triển khai thành code/tool
- giữ context cho ChatGPT, Claude Code, Cursor đọc lại
- tránh mất ý tưởng sau mỗi cuộc trò chuyện
- làm nơi trung gian giữa brainstorm và docs triển khai chính thức

## Các file chính

```text
brainstorm/
├── README.md
├── 2026-05-17-german-ai-learning-os.md
└── claude-handoff.md
```

## Quy ước

- File brainstorm có thể dài, ghi cả ý tưởng chưa hoàn chỉnh.
- File trong `docs/` là bản đã chốt hơn, dùng để triển khai.
- File `brainstorm/claude-handoff.md` dùng để Claude đọc tiếp và đề xuất bước tiếp theo.
