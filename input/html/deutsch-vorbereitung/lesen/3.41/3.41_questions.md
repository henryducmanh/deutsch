# Aufgabe 3.41 — Haustiere willkommen – aber mit Regeln

Source: https://deutsch-vorbereitung.com/en/uebung-13148.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Katzenbesitzer in Lindenthal sollen ihre Tiere tagsüber drinnen halten.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Wer einen Hund besitzt, …

- a) bekommt regelmäßig Futter von der Stadt.
- b) muss in Parks eine Leine benutzen.  **(richtig)**
- c) soll das Tier nachts draußen lassen.

**Lösung:** b) muss in Parks eine Leine benutzen.
