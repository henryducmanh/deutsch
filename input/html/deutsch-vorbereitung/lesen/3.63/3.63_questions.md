# Aufgabe 3.63 — Flohmarkt in der Grundschule

Source: https://deutsch-vorbereitung.com/en/uebung-13070.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Am 15. Mai kann man in der Clara-Schumann-Grundschule günstig Sachen kaufen.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Die Schulleitung möchte,

- a) dass die Eltern Essen und Getränke mitbringen.
- b) dass die Eltern eigene Tische mitbringen.
- c) dass die Eltern eine kleine Gebühr zahlen, wenn sie etwas verkaufen.  **(richtig)**

**Lösung:** c) dass die Eltern eine kleine Gebühr zahlen, wenn sie etwas verkaufen.
