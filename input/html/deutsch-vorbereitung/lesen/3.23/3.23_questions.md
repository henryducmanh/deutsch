# Aufgabe 3.23 — Kostenlos testen – Zeitschrift „Horizonte"

Source: https://deutsch-vorbereitung.com/en/uebung-13166.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Man soll eine Zeitschrift abonnieren.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Wenn man die Zeitschrift „Horizonte“ nicht weiter lesen will, muss man

- a) anrufen oder eine E-Mail schreiben.  **(richtig)**
- b) die beiliegende Karte zurücksenden.
- c) gar nichts unternehmen.

**Lösung:** a) anrufen oder eine E-Mail schreiben.
