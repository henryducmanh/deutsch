# Berufsbezogener Deutschkurs für Migranten

Informationsblatt der VHS Nordstadt
Diese Aufbaukurse wenden sich an Zugewanderte, die ihre Chancen auf dem deutschen Arbeitsmarkt verbessern möchten. Teilnehmen kann man sowohl bei aktiver Arbeitssuche als auch dann, wenn man schon beschäftigt ist und für die aktuelle Stelle zusätzliche Fachsprache benötigt.
Wer sich anmelden will, muss in der Regel einen allgemeinen Integrationskurs abgeschlossen haben. Diese Bedingung entfällt jedoch, wenn Sie nachweisen, dass Sie bereits das Sprachniveau B1 oder höher besitzen.
Im Unterricht trainieren Sie praxisnahes Deutsch für den Beruf, besuchen regionale Betriebe und absolvieren ein Kurzpraktikum.
Die Kurse sind gebührenfrei. Sollten Sie Kinder haben, kann ein Zuschuss für externe Kinder­betreuung beantragt werden. Eine Betreuung im Kursraum ist nicht vorgesehen.
