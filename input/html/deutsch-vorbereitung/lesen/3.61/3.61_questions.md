# Aufgabe 3.61 — Berufsbezogener Deutschkurs für Migranten

Source: https://deutsch-vorbereitung.com/en/uebung-13072.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Kurse darf man auch besuchen, wenn man bereits eine Arbeitsstelle hat.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Um an den Kursen teilnehmen zu können, …

- a) muss man gleichzeitig einen Basiskurs besuchen.
- b) muss man Deutschkenntnisse auf mindestens B1-Niveau nachweisen.  **(richtig)**
- c) darf man seine Kinder während des Unterrichts mitbringen.

**Lösung:** b) muss man Deutschkenntnisse auf mindestens B1-Niveau nachweisen.
