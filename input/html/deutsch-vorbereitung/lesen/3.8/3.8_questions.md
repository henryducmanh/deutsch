# Aufgabe 3.8 — Frühlingsbasar an der Gesamtschule Bergfeld – Elterninformation

Source: https://deutsch-vorbereitung.com/en/uebung-13181.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Schule sammelt Geld für den Frühlingsbasar.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Die Einnahmen vom Basar …

- a) werden für ein neues Schulprojekt genutzt.  **(richtig)**
- b) gehen vollständig an eine Wohltätigkeitsorganisation.
- c) werden unter den beteiligten Klassen aufgeteilt.

**Lösung:** a) werden für ein neues Schulprojekt genutzt.
