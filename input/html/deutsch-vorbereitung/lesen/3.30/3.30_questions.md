# Aufgabe 3.30 — Mitteilung der EnerVolt GmbH

Source: https://deutsch-vorbereitung.com/en/uebung-13159.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Der Strom wird billiger.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Informationen zu den neuen Tarifen …

- a) bekommt man demnächst per Post.  **(richtig)**
- b) gibt es nur im Internet.
- c) kann man ausschließlich telefonisch erfragen.

**Lösung:** a) bekommt man demnächst per Post.
