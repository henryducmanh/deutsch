# Aufgabe 3.11 — Vorteile für junge Kunden – Das „Start+"-Konto der CityBank

Source: https://deutsch-vorbereitung.com/en/uebung-13178.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Bank bietet jungen Leuten ein kostenloses Konto an.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Ab dem nächsten Jahr kann man mit der StartCard …

- a) Rechnungen bezahlen.
- b) eine Bankkarte bekommen.
- c) ein neues Smartphone günstiger kaufen.  **(richtig)**

**Lösung:** c) ein neues Smartphone günstiger kaufen.
