# Aufgabe 3.51 — Unterstützung für Neubürger

Source: https://deutsch-vorbereitung.com/en/uebung-13138.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Das Land Sachsen-Anhalt plant künftig eine halbe Million Euro für Integrationshelfer ein.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Das Ministerium möchte,

- a) dass 700 neue Helfer eingestellt werden.
- b) dass die Projekte besser zusammenarbeiten.  **(richtig)**
- c) dass die Helfer für ihre Arbeit Lohn bekommen.

**Lösung:** b) dass die Projekte besser zusammenarbeiten.
