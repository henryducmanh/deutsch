# Aufgabe 3.27 — Einladung zum Elternabend

Source: https://deutsch-vorbereitung.com/en/uebung-13162.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Einladung informiert die Eltern über die nächste Klassenfahrt.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Wenn die Eltern am 15. 10. zum Elternabend gehen, …

- a) lernen sie eine neue Kollegin von Frau Bauer kennen.  **(richtig)**
- b) müssen sie den unteren Abschnitt nicht ausfüllen.
- c) sollen sie Frau Sommer den unteren Abschnitt geben.

**Lösung:** a) lernen sie eine neue Kollegin von Frau Bauer kennen.
