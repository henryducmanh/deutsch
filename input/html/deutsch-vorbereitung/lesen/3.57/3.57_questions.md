# Aufgabe 3.57 — Einladung zur Mitgliederversammlung

Source: https://deutsch-vorbereitung.com/en/uebung-13132.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Alle Mitglieder eines Vereins sollen zusammenkommen.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Beim Herbstfest …

- a) gibt es Programmangebote.  **(richtig)**
- b) dürfen nur Vereinsmitglieder teilnehmen.
- c) muss man Eintritt bezahlen.

**Lösung:** a) gibt es Programmangebote.
