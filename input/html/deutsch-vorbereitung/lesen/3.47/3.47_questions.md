# Aufgabe 3.47 — Start des Leseförder-Projekts Lesefuchs

Source: https://deutsch-vorbereitung.com/en/uebung-13142.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Das Leseprogramm ist für Eltern und Kinder gedacht.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Die Eltern sollen bei Interesse

- a) ihr Kind selbst im Internet anmelden.
- b) eine Erklärung unterschreiben.  **(richtig)**
- c) von der Lehrkraft ein Passwort verlangen.

**Lösung:** b) eine Erklärung unterschreiben.
