# Aufgabe 3.46 — Stadt Sonnenfeld: Fahrkartenservice für Haushalte mit wenig Geld

Source: https://deutsch-vorbereitung.com/en/uebung-13143.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Stadt Sonnenfeld verkauft günstige Monatskarten an Familien.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Wo kann man die Fahrkarten bekommen?

- a) per Post nach Hause
- b) im Internet bestellen
- c) im Bürgerbüro abholen  **(richtig)**

**Lösung:** c) im Bürgerbüro abholen
