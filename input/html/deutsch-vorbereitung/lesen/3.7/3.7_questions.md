# Aufgabe 3.7 — Information der Stadtverwaltung – Mehr Betreuungsplätze in Sicht

Source: https://deutsch-vorbereitung.com/en/uebung-13182.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

In Falkenau wird bald eine neue Kindertagesstätte eröffnet.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

In Falkenau …

- a) bekommen alle Kinder garantiert einen Betreuungsplatz.
- b) sollte man sein Kind so früh wie möglich anmelden.  **(richtig)**
- c) bewerben sich viele Eltern als Erzieherinnen.

**Lösung:** b) sollte man sein Kind so früh wie möglich anmelden.
