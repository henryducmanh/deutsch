# Aufgabe 3.21 — Warnung der Verbraucherberatung – Abo-Falle per Kurznachricht

Source: https://deutsch-vorbereitung.com/en/uebung-13168.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Meldung informiert über eine Betrugsmasche per SMS.

- a) richtig  **(richtig)**
- b) falsch

**Lösung:** a) richtig

## Frage 2

Wer per SMS zu einem Chat aufgefordert wird, soll …

- a) die Nachricht sofort löschen.  **(richtig)**
- b) eine SMS mit „Stopp“ zurückschicken.
- c) das Wort „OK“ senden.

**Lösung:** a) die Nachricht sofort löschen.
