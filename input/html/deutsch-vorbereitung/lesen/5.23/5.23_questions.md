✅ 6. Richtig: „würde“

… denn ohne gültigen Fahrausweis (6) sie eine Strafe zahlen.
→ … ohne Fahrausweis

sie eine Strafe zahlen.
Begründung:
„würde + Infinitiv“ bildet den
Konjunktiv II
und drückt eine hypothetische Folge/Bedingung aus („falls …, würde …“).
❌
a) könnte
→ Möglichkeit, nicht Folgepflicht.
❌
b) musste
→ Präteritum von „müssen“, Vergangenheit, aber hier Zukunft/Irrealis.
