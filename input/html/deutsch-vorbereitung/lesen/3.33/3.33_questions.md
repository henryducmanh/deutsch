# Aufgabe 3.33 — Aufruf an alle Eltern – Lernwerkstatt 5-7

Source: https://deutsch-vorbereitung.com/en/uebung-13156.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Die Schule sucht Kinder für die Lernwerkstatt.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Die Lernwerkstatt …

- a) findet von Anfang an täglich statt.
- b) wird mit 8 Euro je Stunde bezahlt.
- c) wird auf einer schulischen Info-Veranstaltung näher erklärt.  **(richtig)**

**Lösung:** c) wird auf einer schulischen Info-Veranstaltung näher erklärt.
