# Aufgabe 3.3 — Neues Angebot Ihrer Krankenkasse

Source: https://deutsch-vorbereitung.com/en/uebung-13458.html
Teil: 3 (Teil 3 – Kurze Texte/Mitteilungen)

## Frage 1

Frau Neubert ist neu bei der RheinVital Krankenkasse.

- a) richtig
- b) falsch  **(richtig)**

**Lösung:** b) falsch

## Frage 2

Beim Tarif „AktivPlus“ bezahlt die Krankenkasse:

- a) für alle Kunden ein Wellness-Wochenende.
- b) einen Teil der Beiträge für den Sportverein.  **(richtig)**
- c) Gesundheitskurse vollständig.

**Lösung:** b) einen Teil der Beiträge für den Sportverein.
