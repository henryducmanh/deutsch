# Transcript — 1.84 Kunden-Hotline von Internet NOVA

Source: https://deutsch-vorbereitung.com/en/uebung-1896.html

Die Unterhaltung verursacht keine Kosten für Sie.

Hallo. Herzlich willkommen bei der Kundenhotline von Internet Nova. Wir stehen Ihnen gerne zur Verfügung.

Wenn Sie Fragen zu unseren Tarifen haben, Änderungen an den Tarifen oder ähnliches benötigen, drücken Sie bitte die Erste.

Wenn Sie Fragen zu Ihrer Rechnung haben, drücken Sie bitte die Zweite.

Für technische Unterstützung und Informationen zur Einrichtung von drahtlosen Internetverbindungen, drücken Sie bitte die Dritte.

Wir leiten Sie dann an einen Mitarbeiter weiter.
