# Transcript — 2.91 Im Buchladen

Source: https://deutsch-vorbereitung.com/en/uebung-1064.html

Herzlich willkommen im Buchladen. Beachten Sie bitte, dass aufgrund von Platzmangel im Lager bestimmte Bücher vorübergehend nicht verfügbar sind. Wenn Sie ein bestimmtes Buch suchen, empfehlen wir, vorher anzurufen, um die aktuelle Verfügbarkeit zu klären. Wir danken für Ihr Verständnis und freuen uns auf Ihren Besuch.
