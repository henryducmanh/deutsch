# Transcript — 2.86 NRW-Tour

Source: https://deutsch-vorbereitung.com/en/uebung-1402.html

Herzlich willkommen zu einer weiteren Ausgabe unseres beliebten Gewinnspiels NRW Tour. Diese Woche haben Sie die Chance, eine Kurzreise nach Dortmund zu gewinnen. Genießen Sie eine Verwöhnauszeit in einem Vier-Sterne-Hotel und entdecken Sie die Schönheit der Stadt Dortmund. Wenn Sie teilnehmen möchten, rufen Sie uns einfach an und beantworten Sie unsere Gewinnfrage. Die Telefonnummer wird Ihnen nach einer kurzen Werbepause mitgeteilt.
