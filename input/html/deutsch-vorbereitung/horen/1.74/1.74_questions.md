# Aufgabe 1.74 — Busbahnhof

Source: https://deutsch-vorbereitung.com/en/uebung-1907.html

## Sie wollen mit der S4 zum Busbahnhof. Von welchem Gleis fahren Sie ab?

- a) Von Gleis 4.
- b) Von Gleis 5.
- c) Von Gleis 7.  **(richtig)**

**Lösung:** c) Von Gleis 7.

**Erklärung:**

In der Ansage wird gesagt, dass die Linie S4 zum Busbahnhof heute von Gleis 7 und nicht von Gleis 5 abfährt. Deshalb ist Antwort 3 richtig.

Antwort 1, Von Gleis 4, ist falsch, weil Gleis 4 nicht erwähnt wird. Antwort 2, Von Gleis 5, ist auch falsch, da angegeben wird, dass die S4 heute nicht von Gleis 5, sondern von Gleis 7 abfährt.
