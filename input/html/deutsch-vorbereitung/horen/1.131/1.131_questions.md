# Aufgabe 1.131 — Angebot einer Wohnung

Source: https://deutsch-vorbereitung.com/en/uebung-290.html

## Was für eine Wohnung kann Familie Müller bekommen?

- a) Eine 3-Zimmer-Wohnung.  **(richtig)**
- b) Eine Wohnung mit Balkon.
- c) Eine 2-Zimmer-Wohnung.

**Lösung:** a) Eine 3-Zimmer-Wohnung.

**Erklärung:**

Die richtige Antwort ist 1, da in der Ansage erwähnt wird, dass die angebotene Wohnung eine Drei-Zimmer-Wohnung ist. Option 2 ist falsch, da die Wohnung keinen Balkon hat. Option 3 ist ebenfalls falsch, da es sich nicht um eine Zwei-Zimmer-Wohnung handelt.
