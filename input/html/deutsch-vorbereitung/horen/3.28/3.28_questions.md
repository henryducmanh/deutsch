# Aufgabe 3.28 — Auf der Arbeit

Source: https://deutsch-vorbereitung.com/en/uebung-3969.html

## Die neue Kaffeemaschine bietet kostenpflichtigen Kaffee an.

- a) Richtig
- b) Falsch  **(richtig)**

**Lösung:** b) Falsch

**Erklärung:**

Der Kollege sagt ausdrücklich, dass die neue Kaffeemaschine gratis Kaffee für alle Mitarbeiter bietet.

## Wo wird die neue Kaffeemaschine stehen?

- a) Nur im Erdgeschoss
- b) Im Erdgeschoss und im dritten Stock  **(richtig)**
- c) Nur im dritten Stock

**Lösung:** b) Im Erdgeschoss und im dritten Stock

**Erklärung:**

Der Kollege erklärt, dass eine Maschine in der Küche im Erdgeschoss und die andere im dritten Stock stehen wird.
