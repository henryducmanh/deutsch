# Aufgabe 2.34 — Waldbrandgefahr

Source: https://deutsch-vorbereitung.com/en/uebung-4092.html

## Wo ist das Grillen erlaubt?

- a) An öffentlichen Grillplätzen  **(richtig)**
- b) Auf offenen Feuerstellen
- c) Im Stadtpark

**Lösung:** a) An öffentlichen Grillplätzen

**Erklärung:**

In der Durchsage wird gesagt, dass nur die öffentlichen Grillplätze am Stadtrand genutzt werden dürfen.
