# Aufgabe 1.94 — Fitnessstudio - eine kleine Party zum Jubiläum

Source: https://deutsch-vorbereitung.com/en/uebung-1834.html

## Warum ist das Fitnessstudio geschlossen?

- a) Wegen einer Renovierung.  **(richtig)**
- b) Wegen einer Krankheit.
- c) Wegen einer Party.

**Lösung:** a) Wegen einer Renovierung.

**Erklärung:**

Im Text steht: "Wegen Renovierungsarbeiten müssen wir das Studio schließen."
Das bedeutet: Es wird etwas im Studio gebaut oder repariert.
Die Krankheit und die Party werden nur erwähnt – aber das ist nicht der Grund für die Schließung.
Deshalb ist Antwort a richtig.
