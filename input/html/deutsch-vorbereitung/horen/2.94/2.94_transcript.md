# Transcript — 2.94 Blumensträuße

Source: https://deutsch-vorbereitung.com/en/uebung-1061.html

Herzlich willkommen in Blumenladen. Aufgrund von Personalengpässen können wir derzeit keine individuellen Blumenstreuse zusammenstellen. Wir bieten jedoch eine Auswahl vorbereiteter Streuse an. Wenn Sie spezielle Wünsche haben, empfehlen wir, vorher anzurufen, um die Verfügbarkeit zu klären. Wir danken für Ihr Verständnis.
