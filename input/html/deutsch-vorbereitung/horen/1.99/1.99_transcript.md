# Transcript — 1.99 Gesundheitszentrum

Source: https://deutsch-vorbereitung.com/en/uebung-1654.html

Guten Tag, hier ist Ihr örtliches Gesundheitszentrum. Aufgrund von anstehenden Renovierungsarbeiten wird der Zugang zum Fitnessraum vorübergehend eingeschränkt. Für Mitglieder besteht die Möglichkeit, an den Gruppenkursen im Park teilzunehmen. Für genauere Informationen zu alternativen Trainingsmöglichkeiten stehen Ihnen unsere Mitarbeiter im Gesundheitszentrum gerne zur Verfügung.
