# Aufgabe 2.114 — Expresszug nach Berlin

Source: https://deutsch-vorbereitung.com/en/uebung-957.html

## Welche Information wird den Passagieren mitgeteilt?

- a) Der Zug fährt verspätet ab.
- b) Der Zug wird eine andere Strecke nehmen als üblich.  **(richtig)**
- c) Die Ankunft des Zuges wurde vorverlegt.

**Lösung:** b) Der Zug wird eine andere Strecke nehmen als üblich.

**Erklärung:**

In der Nachricht wird den Passagieren mitgeteilt, dass der Zug abweichend von der normalen Route fahren wird, was darauf hinweist, dass eine andere Strecke genommen wird. Es wird nicht gesagt, dass der Zug verspätet abfährt oder die Ankunft vorverlegt wurde, weshalb diese Antworten nicht zutreffen.
