# Aufgabe 2.31 — Schwimmturnier

Source: https://deutsch-vorbereitung.com/en/uebung-4095.html

## Was ist morgen im Freibad geschlossen?

- a) das Kinderbecken
- b) die Liegewiese
- c) das Sportbecken  **(richtig)**

**Lösung:** c) das Sportbecken

**Erklärung:**

In der Durchsage wird erwähnt, dass das Sportbecken ganztägig geschlossen ist.
