# Aufgabe 2.33 — Sonnenschein

Source: https://deutsch-vorbereitung.com/en/uebung-4093.html

## Was bietet die Apotheke an?

- a) Rabatt auf Medikamente
- b) Gesundheitsberatung  **(richtig)**
- c) kostenlose Impfungen

**Lösung:** b) Gesundheitsberatung

**Erklärung:**

In der Durchsage steht, dass es eine kostenlose Gesundheitsberatung zum Thema „Erkältung vorbeugen“ gibt.
