# Transcript — 2.71 Wochenende nach Frankreich

Source: https://deutsch-vorbereitung.com/en/uebung-1945.html

Sie wollen übers Wochenende nach Frankreich fahren? Lux FM Radio macht es möglich. Rufen Sie uns an und sagen Sie uns, welchen Song Sie gerade mögen. Welcher Song sollte in den Charts sein? Mit etwas Glück können Sie eine Woche Urlaub in Paris für zwei Personen in einem Fünf-Sterne-Hotel gewinnen. Außerdem gibt es jede Woche zehn Einkaufsgutscheine im Wert von 150 Euro zu gewinnen. Rufen Sie also an und gewinnen Sie mit Lux-FM Radio.
