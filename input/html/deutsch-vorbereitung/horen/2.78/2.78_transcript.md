# Transcript — 2.78 Eine dringende Ankündigung

Source: https://deutsch-vorbereitung.com/en/uebung-1938.html

Unser reguläres Programm wird für eine dringende Ankündigung unterbrochen. Aufgrund eines schwerwiegenden Verkehrsunfalls ist der Bereich um den Hauptbahnhof bis zum Abend für den Autoverkehr vollständig gesperrt. Zusätzlich verkehren keine Straßenbahnen und Busse zum Hauptbahnhof. Falls Sie dennoch zum Bahnhof müssen, nutzen Sie bitte die U- und S-Bahn.
