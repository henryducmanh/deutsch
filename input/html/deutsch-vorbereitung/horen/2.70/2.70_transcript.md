# Transcript — 2.70 Vermisst

Source: https://deutsch-vorbereitung.com/en/uebung-1946.html

Hier sind die aktuellen Nachrichten. Zusätzlich liegt uns eine Bekanntmachung der Polizei vor. Seit gestern Abend wird Herr Müller vermisst. Herr Müller, 80 Jahre alt, begab sich auf dem Heimweg von der Kaiserstraße zu seiner Wohnung im Kreuzviertel von Dortmund. Er ist etwa 1,70 Meter groß, trägt einen braunen Wintermantel und eine Brille. Aufgrund seiner Orientierungsschwierigkeiten bitten wir um Hinweise, die von jeder Polizeidienststelle entgegengenommen werden.
