# Aufgabe 2.61 — Lux FM

Source: https://deutsch-vorbereitung.com/en/uebung-1955.html

## Auf der Silvesterparty

- a) treten Gruppen aus verschiedenen Ländern auf.  **(richtig)**
- b) gibt es nur Musik in deutscher Sprache.
- c) kann man Preise gewinnen.

**Lösung:** a) treten Gruppen aus verschiedenen Ländern auf.

**Erklärung:**

Die richtige Antwort ist Option 1 ("treten Gruppen aus verschiedenen Ländern auf"), weil in der Ansage steht: "auf der World-Bühne können Sie Musiker aus Afrika und Asien erleben."

Option 2 ist falsch, da auf der Deutsch-Pop-Bühne ausschließlich deutsche Musik gespielt wird, aber auch andere Musikrichtungen auf anderen Bühnen angeboten werden.
Option 3 ist ebenfalls falsch, da es zwar eine Verlosung gibt, aber das Hauptthema der Frage auf die auftretenden Musiker abzielt.
