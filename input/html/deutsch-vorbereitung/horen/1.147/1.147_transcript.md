# Transcript — 1.147 Wer ruft Burak an

Source: https://deutsch-vorbereitung.com/en/uebung-68.html

Hier spricht das Büro von Dr. Müde. Herr Burak, gestern haben Sie telefonisch ein neues Rezept bestellt. Das Rezept wurde bereits ausgestellt und liegt hier für Sie bereit. Wir würden uns freuen, wenn Sie kurz bei uns vorbeikommen könnten. Denken Sie bitte daran, Ihre Versichertenkarte mitzubringen.
