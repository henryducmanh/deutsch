# Transcript — 2.108 Sortiment

Source: https://deutsch-vorbereitung.com/en/uebung-1046.html

Achtung, alle Kunden! Wir informieren Sie darüber, dass aufgrund von erhöhter Nachfrage einige Produkte in unserem Sortiment vorübergehend ausverkauft sind. Wir erwarten eine Lieferung in der nächsten Woche. Sie können jedoch eine Vorbestellung aufgeben, um sicherzustellen, dass Sie Ihr gewünschtes Produkt erhalten, sobald es wieder verfügbar ist.
