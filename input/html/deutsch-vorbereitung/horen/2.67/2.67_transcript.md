# Transcript — 2.67 Recht und Finanzen

Source: https://deutsch-vorbereitung.com/en/uebung-1949.html

Bis zu welchem Zeitpunkt können Sie Ihre bestehende Versicherung kündigen? Im Anschluss folgen aktuelle Nachrichten aus der Rechtswelt. Bleiben Sie auf Empfang.
