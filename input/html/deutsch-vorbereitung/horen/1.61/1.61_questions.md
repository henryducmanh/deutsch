# Aufgabe 1.61 — Aufruf

Source: https://deutsch-vorbereitung.com/en/uebung-1920.html

## Wer ruft an?

- a) Eine Apotheke.
- b) Eine Arztpraxis.  **(richtig)**
- c) Eine Versicherung.

**Lösung:** b) Eine Arztpraxis.

**Erklärung:**

Im Text stellt sich Klara Schulz vor und sagt, dass sie von der Praxis Dr. Krantz anruft. Dies zeigt eindeutig, dass der Anruf von einer Arztpraxis kommt.

Die anderen Optionen sind nicht korrekt, weil:

Eine Apotheke: Im Text wird keine Apotheke erwähnt. Der Anruf erfolgt direkt von einer Arztpraxis und nicht von einer Apotheke.

Eine Versicherung: Obwohl die Versichertenkarte erwähnt wird, geschieht dies im Kontext der Arztpraxis, nicht einer Versicherung. Der Anruf ist nicht von einer Versicherung.

Zusammenfassend ruft eine Arztpraxis an.
