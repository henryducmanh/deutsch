# Transcript — 2.106 Wetterbedingungen

Source: https://deutsch-vorbereitung.com/en/uebung-1048.html

Guten Tag. Wir möchten Sie darauf hinweisen, dass aufgrund von schlechten Wetterbedingungen für die nächsten Tage Regen und Gewitter erwartet werden. Wir empfehlen, entsprechende Vorbereitungen zu treffen, wenn Sie im Freien unterwegs sind. Überprüfen Sie den detaillierten Wetterbericht auf unserer Website für aktuelle Informationen.
