# Transcript — 2.109 Mitteilung

Source: https://deutsch-vorbereitung.com/en/uebung-1045.html

Guten Tag. Dies ist eine wichtige Mitteilung für alle Bibliotheksbenutzer. Beachten Sie bitte, dass aufgrund von Renovierungsarbeiten die Bibliothek ab dem nächsten Montag für zwei Wochen geschlossen sein wird. Während dieser Zeit stehen jedoch digitale Ressourcen online zur Verfügung. Überprüfen Sie unseren Online-Katalog für den Zugang zu E-Books und Audiobüchern.
