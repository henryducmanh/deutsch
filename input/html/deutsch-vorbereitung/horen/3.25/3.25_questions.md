# Aufgabe 3.25 — Beim Friseur

Source: https://deutsch-vorbereitung.com/en/uebung-3974.html

## Herr Meier war bereits beim Arzt wegen seines Knies.

- a) Richtig
- b) Falsch  **(richtig)**

**Lösung:** b) Falsch

**Erklärung:**

Herr Meier sagt, dass er noch nicht beim Arzt war, aber gehen möchte, wenn es nicht besser wird.

## Wo findet der Dialog statt?

- a) In einer Klinik
- b) Im Wartezimmer eines Orthopäden
- c) Beim Friseur  **(richtig)**

**Lösung:** c) Beim Friseur

**Erklärung:**

Obwohl das Thema Gesundheit besprochen wird, geht es anschließend um einen Haarschnitt. Das zeigt, dass der Dialog im Friseursalon stattfindet.
