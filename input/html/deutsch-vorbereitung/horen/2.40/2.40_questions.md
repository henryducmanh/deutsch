# Aufgabe 2.40 — Verkehrsprobleme

Source: https://deutsch-vorbereitung.com/en/uebung-4086.html

## Wo gibt es aktuell keine Verkehrsprobleme?

- a) auf der A2 Südautobahn
- b) auf der A10 Tauernautobahn  **(richtig)**
- c) auf der A1 Westautobahn

**Lösung:** b) auf der A10 Tauernautobahn

**Erklärung:**

In der Durchsage wird gesagt, dass der Verkehr auf der A10 Tauernautobahn normal fließt.
