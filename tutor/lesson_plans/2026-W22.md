## Buổi gia sư 1.5h/tuần — GIÁO ÁN AI THIẾT KẾ

**Trước đây:** gia sư dạy theo cảm tính → input thụ động → phí 1.5h.

**Bây giờ:** AI gửi giáo án cho user mang đi học gia sư. Cấu trúc 90 phút:

| Phút | Hoạt động | Mục đích |
|---|---|---|
| 0-10 | **Mini-test recall**: gia sư đọc 10 cảnh huống Việt → user trả lời tiếng Đức ngay (không ghi) | đo retrieval speed |
| 10-25 | **Sprechen Teil 3 (Planung)**: roleplay 1 chủ đề DTZ (vd: "lên kế hoạch tiệc sinh nhật cho bạn") — gia sư đóng vai đối thoại | drill production |
| 25-45 | **Schreiben review**: user nộp 3 Brief đã viết trong tuần → gia sư sửa từng câu, đánh dấu lỗi grammar lặp lại | identify pattern lỗi |
| 45-55 | **Phát âm + intonation**: 5 câu user nói trong Sprechen — gia sư correct phát âm | sound mapping |
| 55-75 | **Hören "chủ đề mới"**: gia sư bật 1 audio user CHƯA nghe bao giờ → cùng listen → giải thích từ user miss | audio adaptation real-time |
| 75-85 | **Q&A grammar**: user mang 3 câu grammar đang vướng (vd Konjunktiv II, Passiv) → gia sư giải | targeted, không lan man |
| 85-90 | **Đặt mục tiêu tuần sau**: gia sư chốt 2-3 chủ đề cần focus | continuity |

**Output sau gia sư** (gia sư hoặc user ghi):
- File `tutor/gia_su_<YYYY-MM-DD>.md`: lỗi lặp, từ mới gia sư dùng, chủ đề tuần sau
- Append `data/weak_words.csv` với từ mới gia sư correct
