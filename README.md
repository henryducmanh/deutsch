# deutsch

Kho học tiếng Đức tối giản.

## Mục tiêu
Chỉ giữ 2 loại dữ liệu cốt lõi:

1. Cặp:
   - `Từ hoặc cụm từ tiếng Đức`
   - `Nghĩa tiếng Việt`

2. Mức độ học:
   - `Từ hoặc cụm từ tiếng Đức`
   - mức độ `1,2,3,4,5`

## Quy ước mức độ học
- `1`: mới
- `2`: nhận biết
- `3`: làm quen
- `4`: đã học
- `5`: đã biết
