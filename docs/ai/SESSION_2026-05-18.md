## 3 Grammar Point (must-review)

### 1. Passiv + Modalverb (đa biến thể)

Pattern căn bản: `Subjekt + Modal + Partizip II + werden`

- `Künstliche Intelligenz sollte ethisch gestaltet werden.` *(sollte = nên)*
- `Soziale Ungerechtigkeiten dürfen nicht verstärkt werden.` *(dürfen + Negation = bị cấm)*
- `Die Fähigkeiten müssen vermittelt werden.` *(müssen = phải)*

Variant Konsekutiv (B1+): `... sollte so + P.II + werden, dass + Nebensatz`
→ `Das Programm sollte so gestaltet werden, dass alle es benutzen können.` *(CH-013)*

⚠️ Word order: trong Hauptsatz, `werden` đứng cuối; trong Nebensatz, cả Modal + P.II + werden xếp về cuối: `…, dass das Programm so gestaltet werden sollte.`

### 2. Verb/N + Präposition + fester Kasus (DTZ critical)

DTZ test phần điền chỗ trống hay test giới từ + Kasus. Bảng nhớ buổi này:

| Pattern | Kasus | Ví dụ |
|---|---|---|
| `sich an + Akk. anpassen` | Akk. | sich an die digitalisierte Welt anpassen |
| `sich auf + Akk. einstellen` | Akk. | sich auf neue Aufgaben einstellen |
| `auf + Akk. verzichten` | Akk. | auf tierische Produkte verzichten |
| `Einfluss auf + Akk. haben` | Akk. | großen Einfluss auf die Gesellschaft haben |
| `Bewusstsein für + Akk. schaffen` | Akk. | Bewusstsein für Umweltschutz schaffen |
| `mit + Dat. verbunden sein` | Dat. | mit dem WLAN verbunden sein |
| `mit + Dat. aufwachsen` | Dat. | mit Technologie aufwachsen |
| `führen zu + Dat.` | Dat. | zu Frust und Scheitern führen |
| `basierend auf + Dat.` | Dat. | basierend auf persönlichen Bedürfnissen |

Mẹo nhớ: `auf` có thể Akk. (direction/ziel) HOẶC Dat. (location/Zustand) — học theo từng V/N cố định, không suy luận.

### 3. Konjunktionen + Nebensatz-Pattern (B1 connectors)

| Connector | Funktion | Ví dụ buổi này |
|---|---|---|
| `nicht nur ..., sondern auch ...` | additive (2 vế song hành) | nicht nur für die Gesundheit, sondern auch für die Umwelt |
| `Es ist + Adj, dass + V-final` | impersonal evaluation | Es ist entscheidend, dass Bildungssysteme sich anpassen |
| `um ... zu + Infinitiv` | Finalsatz (mục đích) | um in einer digitalisierten Welt erfolgreich zu sein |
| `jdm. helfen, ... zu + Inf.` | helfen + Dat. + Inf.satz | helfen, gesündere Entscheidungen zu treffen |
| `deren / dessen + N` | Genitiv-Relativpronomen | Technologie und deren Einfluss |
| `so ..., dass + V-final` | Konsekutivsatz | so gestaltet werden, dass... |

⚠️ Phân biệt **`deren` vs `dessen`** *(Genitiv-Relativ)*:
- `deren` → tiền tố là **f.Sg.** hoặc **Pl.** *(các Entwicklungen → deren)*
- `dessen` → tiền tố là **m.Sg.** hoặc **n.Sg.** *(der Mann → dessen / das Auto → dessen)*

## 5 Từ key (chú ý kẻo nhầm)

1. **schaffen** *(VOC-055)* — **2 paradigm phân theo nghĩa**:
   - `schuf — hat geschaffen` (irreg.) = tạo ra, sáng tạo *(kreieren)*
   - `schaffte — hat geschafft` (reg.) = làm được, hoàn thành *(es schaffen)*
   - B1 cực hay sai chia.

2. **deren / dessen** *(CH-009)* — Genitiv-Relativ phân theo giới tiền tố. Học bảng (xem grammar point 3).

3. **vermitteln** *(VOC-028)* — **Collocation học thuật**: `Wissen / Fähigkeiten / Kenntnisse vermitteln`. Phụ nghĩa: làm trung gian/hoà giải.

4. **Bewusstsein** *(VOC-054)* + **schaffen** *(VOC-055)* — Collocation B1: `Bewusstsein für + Akk. schaffen / wecken / schärfen` *(CH-022)*.

5. **verzichten auf + Akk.** *(VOC-056 + CH-019)* — fixed prep. Đừng nhầm với `vermeiden` *(Akk. trực tiếp, không prep)*.
